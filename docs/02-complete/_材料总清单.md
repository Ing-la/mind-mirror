## 二、需要你提供的素材

### 2.1 截图素材（至少 3 张，建议 5-6 张）

以下分类的截图，你任意截好后放到 `raw/` 文件夹，告诉我是哪类即可：

**产品展示截图（放在 Demo 简介部分）：**

- [ ] 首页/欢迎页 — 展示 MindMirror 品牌调性
- [ ] 新建对话页 — 展示输入题目 + 选角流程
- [ ] 对话剧场页 — 展示两个小人对话的实时画面（最核心的截图！）
- [ ] 自定义角色功能 — 展示用户可以自定义小人性格

**TRAE 开发过程截图（放在 TRAE 实践部分）：**

- [ ] TRAE IDE 中编写/生成关键代码的画面（如对话引擎逻辑）
- [ ] TRAE 对话窗口 — 展示你如何通过对话让 AI 生成代码
- [ ] 修改/调试过程 — 展示你通过 TRAE 修复 bug 或迭代功能

**部署相关截图（可选，加分项）：**

- [ ] Vercel 部署成功页面
- [ ] 项目在浏览器中运行的最终效果

### 2.2 Session ID（至少 3 个）

在 TRAE IDE 中双击对话任务复制 Session ID，发给我。建议涵盖：

- [ ] Session 1：最初生成对话引擎核心逻辑的对话
- [ ] Session 2：迭代角色设定 / UI 界面的对话
- [ ] Session 3：修复某个 bug 或添加某个功能的对话
- [ ] （可选）更多 Session 更有说服力

### 2.3 文案信息（可选）

如果你有想补充的：

- [ ] 开发过程中有没有印象深刻的"翻车"经历？（写进帖子里很加分）
- [ ] 有没有特别得意的 Prompt 想展示？
