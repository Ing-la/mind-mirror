### Demo MVP 完成

**核心对话**

- 首页（品牌 + 历史列表，localStorage）+ 新建对话（双角色选角）
- 对话剧场：DeepSeek 流式输出、半自动对话、用户插话、观点摘要、10 轮截停

**心声系统**

- 完整 CRUD：`/voices` 列表 → `/voices/new` 创建 → `/voices/[id]/edit` 编辑
- 创建表单：名字 + 灵魂设定 + 9 色头像选择 + 大脑（API）配置

**大脑（API）配置**

- 内置默认：服务端 env var，Key 不暴露到浏览器
- 自定义 API：用户配置地址/Key/模型，存 localStorage，各角色可绑定不同大脑

**角色头像**

- SVG 手绘小人组件，4 种表情 × 9 种颜色

**API 路由**：`/api/chat` 接收 `brain` 参数，自定义 endpoint/model/Key 透传到 LLM

**架构决策**

- 无登录，全部 localStorage
- 4 个预设角色硬编码，自定义角色存 localStorage
- 双 LLM 实例交替发言，各自仅知道自己的角色设定 + 完整对话历史
